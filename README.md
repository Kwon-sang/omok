# Omok

> CMD 콘솔 기반 오목 게임

<br><br>

## 1. 모듈 구조

* `main.py` : 게임 실행 로직<br>
* modules
    * `settings.py` : 바둑판 마킹, 입력 메시지, 오류 메시지 등 **텍스트 변수**를 관리.
    * `commands.py` : 사용자 입력 관련 메서드 정의.
    * `validations.py` : 입력값 검증을 위한 메서드 정의.
    * `boards.py` : 바둑판 생성, 포석, 출력, 오목검증 등 바둑판 관련 메서드 정의.

<br>

 ## 2. 기능 구현
 * 바둑판 크기 입력은 5~19 사이의 숫자
 * 포석 위치는 공백(space)로 구분하여 입력. ex) '3 4'
   * 입력값은 숫자+공백+숫자로 제한
   * 각 숫자는 바둑판 크기 내로 제한
   * 'q' or 'Q' 입력 시 게임 종료
   * 잘못된 입력의 경우, 에러메시지 출력 및 입력창 반복
* 오목검증 로직은 각 사용자의 마지막 포석 위치 기준으로 오목검증 로직을 수행
* 커맨드 UI를 위한 clear 적용
